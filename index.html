Auth ¶

This section groups Curio Nest auth resources.

Csrf  ¶

CSRF（Cross-Site Request Forgeries）攻撃に対する対策として、 csrfTokenを発行します。

以後、RequestのHeaderにcsrfTokenが含まれない場合は、不正なリクエストとみなされます。

Retrieve Csrf Token
GET/auth/csrf

Csrf Tokenを返します。

Example URI

GET https://curio-nest-server.onrender.com/auth/csrf
Response  200
HideShow
Headers
Content-Type: application/json
Body
{
  "csrfToken": "LUTuhj0x-50aQ4F7_sHT-vZlInx5_sCMZM9U"
}

Login  ¶

Login
